## App name

TermKey IME

## Short description

Terminal keyboard with compact Chinese input, macros, and streaming voice input.

## Full description

TermKey IME is an Android keyboard built for terminal users who also need fast everyday text input.

It combines a terminal-first full layout with compact Chinese and English layouts, so you can switch between shell work, command entry, and normal typing without changing keyboards.

Key features:

- Full terminal keyboard with sticky Ctrl, Alt, Shift, function keys, arrows, PgUp, PgDn, Home, and End
- Compact Chinese layout with Natural Shuangpin input
- Offline lexicon-backed Chinese candidates with phrase and sentence-first ranking
- Local frequency learning for faster candidate ordering over time
- Compact English layout with numbers and common punctuation
- Dedicated symbol mode for fast punctuation and symbols
- Editable macro bar for common commands and snippets
- Streaming voice input with optional Volcengine speech recognition
- Hold-to-delete, swipe-to-clear, and enlarged touch targets in compact mode

TermKey IME works especially well with terminal and SSH apps such as Termux, ConnectBot, JuiceSSH, and Blink Shell, while still being usable in normal Android text fields.

Voice input is optional and requires microphone permission plus your own speech provider configuration.
