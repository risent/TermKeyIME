TermKey IME Privacy Policy

Last updated: March 10, 2026

TermKey IME is an Android input method for terminal-style input, compact Chinese and English typing, local macro insertion, and optional voice input.

Data We Process

TermKey IME does not require an account and does not operate its own backend service.

  • Text you type through the keyboard
  • Locally stored macro definitions
  • Locally stored Chinese input frequency data
  • Optional voice audio when you explicitly start voice input
  • Optional voice recognition text returned by the configured speech provider

Voice Input

Voice input is optional.

If you enable voice input and configure Volcengine speech recognition:

  • Microphone audio is sent to the speech recognition provider you configured
  • Recognized text is returned to the app and inserted into the active text field
  • TermKey IME itself does not provide or operate the speech backend

You are responsible for reviewing the privacy terms of the configured speech provider.

Local Storage

  • Macro bar configuration
  • Chinese input user frequency data
  • App preferences, such as keyboard height and feature toggles
  • Optional voice input configuration values that you enter in settings

This data stays on device unless you use Android backup or another external device-level sync mechanism.

Data Sharing

TermKey IME does not sell personal data.

The app does not share typed content with the developer's own servers.

If you use optional voice input, audio and recognition-related data are sent to the third-party provider you configured in the app.

Permissions

  • RECORD_AUDIO for optional voice input
  • INTERNET for optional voice recognition requests
  • VIBRATE for haptic feedback

Security, Children, and Contact

TermKey IME stores settings and local input data using normal Android app storage mechanisms. No security method can guarantee absolute protection.

TermKey IME is not specifically directed to children.

For privacy questions, use the public contact channel listed in the app store listing or repository.
